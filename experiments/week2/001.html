Bootstrap - Responsive Fixed Navigation Bar

Description:

This experiment uses the Bootstrap Framework to implement an Responsive Navigation Bar which fixed at the top of the web page.

Experiment Detail:

Basically it make uses of pre-defined css class and javascript in Bootstrap framework, by including Bootstrap from a CDN, and add a padding-top for the body. So the content won't be cover by the nav bar. It has the following features:

1. The Nav Bar stick at the top of the web page, no matter how user scroll the web page.

2. If browser window is too small to display the whole bar, the nav bar will be replaced by an menu icon. And user can use the menu icon to open it.

- Code Snippet:

<nav class="navbar navbar-default navbar-fixed-top">
    <div class="container">
        <div class="navbar-header"> 
            <button type="button" class="navbar-toggle collapsed" data-toggle="collapse" data-target="#myNavbar"> 
                <span class="icon-bar"></span> 
                <span class="icon-bar"></span> 
                <span class="icon-bar"></span> 
            </button> 
            <a class="navbar-brand" href="#">ZhenminG's</a> 
        </div> 
        <div id="myNavbar" class="collapse navbar-collapse"> 
            <ul class="nav navbar-nav"> 
                <li><href="#" target="_blank">Experiements</a></li> 
                <li><href="#" target="_blank">Project</a> </li>
                <li><href="#" target="_blank">Documentation</a></li> 
            </ul>
        </div>
     </div>
</nav>
