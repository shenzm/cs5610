Bootstrap - Table

Description:

This experiment use the Bootstrap Framework to create an well formed table

Experiment Detail:

Basically it make uses of pre-defined css class in Bootstrap framework, by including Bootstrap from a CDN.

- Code Snippet:

1. Create a table with class table table-hover. So when user move their move over a certain table row, the row will become yellow background color.

2. Certain background color can be set for a table row to give informative tips for the viewer ralted to the table row. As we set the <tr> to class success, the table row will become green background. 

<table class="table table-hover">
    <thead>
        <tr> 
            <th>ID</th> 
            <th>Course</th> 
            <th>Teacher</th> 
            <th>Credit</th> 
            <th>Cost</th> 
        </tr> 
    </thead> 
    <tr class="success">
        <td>sp7777</td>
        <td>Soccer</td>
        <td>Robin Van Persie</td>
        <td>4</td>
        <td>10000</td>
    </tr>
    <tr class="warning">
        <td>sp8888</td>
        <td>Basketball</td>
        <td>Stephen Curry</td>
        <td>4</td>
        <td>10000</td>
    </tr>
</table>
