Bootstrap - Sign In Page

Description:

This experiment uses the Bootstrap Framework and some customized css to implement a login page.

Experiment Detail:

Basically it make uses of pre-defined css class in Bootstrap framework, by including Bootstrap from a CDN. And build up the sign in form in customized css file. User can choose "Remember Me" options as well when they are logging in.

- Code Snippet:

1. Create a <form> for the sign in page.

2. Add two <input> inside the form, one for username, another for password, and provide placeholder for these two <input> tag.

3. Add a <input> that contains a checkbox, so user can choose to store the username and password temporarily in local browser storage.

4. Add a <input> at the end, to enable form submitting for the sign in page. 

<form id="signin-form" class="zhenmins-signin-form">
    <p class="zhenmins-signin-heading">Please Sign In</p>
    <input id="input-email" type="email" class="form-control" 
              placeholder="Email address" required />
    <div class="input-line-br"></div> 
    <input id="input-password" type="password" class="form-control"
              placeholder="Password" required/>
    <div class="input-checkbox"> 
        <label> 
            <input type="checkbox" value="remember-me" />Remember me 
        </label> 
    </div> 
    <button class="btn btn-default btn-block" type="submit">Sign In</button>
</form>
