CSS - Grid System

Description:

This experiment creates a grid system by imitating grid system of Bootstrap Framework. Basic idea is to define different versions of "col-1" to "col-12" for various screen size. These "col" class are prefixed with "xs"(extra small), "sm"(small), "md"(medium),"lg"(large), which means their rules are going to apply on certain screen size.

Experiment Detail:

Follows is the experiment itself, which we use an iframe to display. You can check that different styles applied according to your browser's width. Following are the rules for different class prefix

Prefix Background Color Division
zhenmins-col-xs- lightyellow 12:12:12
zhenmins-col-sm- lightsalmon 4:4:4
zhenmins-col-md- lightblue 3:6:3
zhenmins-col-lg- lightgreen 2:8:2

- Code Snippet:

Use @media selector and condition (min-width) to make the css rule applys on different width of the browser. 

.zhenmins-container {
    padding-left: 15px;
    padding-right: 15px;
    margin-left: auto;
    margin-right: auto;
}
@media (min-width: 768px) {
    .zhenmins-container {
        width: 750px;
    }
}
@media (min-width: 992px) {
    .zhenmins-container {
        width: 970px;
    }
}
@media (min-width: 1200px) {
    .zhenmins-container {
        width: 1170px;
    }
}

.zhenmins-col-xs-12 {
    width: 100%;
    background-color: lightyellow;
}
@media (min-width: 768px) {  
    .zhenmins-col-sm-12 {
        background-color: lightsalmon;
        width: 100%;
    }
}
@media (min-width: 992px) {  
    .zhenmins-col-md-12 {
        background-color: lightblue;
        width: 100%;
    }
}
@media (min-width: 1200px) {  
    .zhenmins-col-lg-12 {
        background-color: lightgreen;
        width: 100%;
    }
}
