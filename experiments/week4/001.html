Bootstrap - Style refinement for experiment pages

This experiment use Bootstrap to refine the style of all experiment page, so as to provide more user-friendly page layout, which includes code highlighting feature

Experiment Detail:

  1. Title and brief introduction of the experiment is displayed in a .jumbotron
  2. Experiment Details and Code Analysis showed in a paragraph inside of .experiment-items.
  3. Code analysis includes well formatted and highlighted snippets, which is easier to read.

Code Analysis:

How to display well-formated and highlighted html code in a webpage?

  1. Download and include the prism.css and prism.js into the web page. 
    <link rel="stylesheet" href="prism/css/prism.css">
<script src="prism/js/prism.js"></script>
  2. Using HTML Escape Formatter to escape the HTML source code, and enclose it into <pre><code>escaped-html</code></pre>. 
    <pre><code class="language-markup scroll-overflow">&lt;div&gt;
    &lt;h3&gt;Example&lt;/h3&gt;
    &lt;p&gt;This is an example&lt;/p&gt;
&lt;/div&gt;</code></pre>
  3. Set the <code> using .language-markup, which tells prism to render the code as a markup lanuage
  4. Define a class named .scroll-overflow, which disable word break and force code to scroll on overflow. And apply this class to code. 
    .scroll-overflow {
    overflow: scroll ;
    word-wrap: normal;
}
