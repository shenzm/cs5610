AngularJS - Event Detail Image View

This experiment builds an single-page app, where user can see different events detail infomation, as well as the large event image by hover mouse at the small event thumbnail.

Experiment Description:

  1. This app is on ImageView.html, which is shown in the following iframe. Its inspired by the product detail page of amazon.com, on which user can hover mouse on different product thumbnail, and see the larger image in the main frame.
  2. User can hover mouse on the thumbnail of NFL super bowl on the right panel, to see the larger image on the left side main frame. And the event informations will also be changed accordingly on the right panel.

Experiment:

Click button on top-left corner to view in new page

Code Analysis:

- HTML Snippet
<div class="imageViewer">
    <div class="col-sm-4 col-xs-6">
        <img ng-src="{{mainEvent.imgUrl}}" class="mainImage" />
    </div>
    <div class="col-sm-8 col-xs-6">
        <div>
            <dl>
                <dt>Event:</dt>
                <dd>{{mainEvent.eventName}}</dd>
                <dt>Time:</dt>
                <dd>{{mainEvent.time}}</dd>
                <dt>Event:</dt>
                <dd>{{mainEvent.place}}</dd>
            </dl>
            <ul class="list-inline thumbnails">
                <li ng-repeat="e in events">
                    <img class="thumbnail" ng-src="{{e.imgUrl}}"
                         ng-mouseenter="setMainEvent(e)" />
                </li>
            </ul>
        </div>
    </div>
</div>
- Javascript Snippet
var app = angular.module("ImageView", []);
app.controller("ImageViewController",
function ($scope) {
    $scope.events = initEvents;
    $scope.mainEvent = initEvents[0];
    $scope.setMainEvent = function (e) {
        $scope.mainEvent = e;
    };
});
  1. Include angular.min.js and ImageView.js as usual.
  2. For <img> with class mainImage, bind the ngSrc directive to the mainEvent.imgUrl property. MainEvent will be initially set to the first element of initEvents in ImageView.js.
  3. For <dl>, we use all the attributes of mainEvent as proterpies to set up a template for the event detail information field.
  4. For the <ul> with class thumbnails, bind ng-repeat with every element e in events, which is set equal to the initEvent in ImageView.js. And for the <li> with class thumbnail, bind the ngSrc directive to the e.imgUrl property, and ng-mouseenter to the function setMainEvent(e), which sets the mainEvent to e.
