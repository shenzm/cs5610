Week10 - Experiment1

UI - Redo Application Landing Page

This experiment focus on using bootstrap templates and tool kits to redesign the landing page of our application. It also add a footer for on the page which shows across all the views.

Experiment Description:

  1. As shown in the following iframe, my application has a simple home page, which include a simple description of itself.
  2. We also defines a general footer shows across all the views, it basically shown the authorship of the webpage.

Experiment:

Click button on top-left corner to view in new page

Operation Analysis:

- HTML Snippet (home.html) 

<div class="landing-header">
    <h1 class="landing-header-title">Application Name</h1>
    <p> Single-page application design in CS5610 - Web Development </p>
    <p> by Zhenming Shen </p>
</div>
<div class="container">
    <div class="col-md-8 col-md-offset-2 col-sm-8 col-sm-offset-2">
        <div class="landing-info">
            <h1 class="landing-info-title">Lorem Ipsum</h1>
            <p class="text-muted">
                Lorem Ipsum is simply dummy text of the printing and typesetting industry.
                Lorem Ipsum has been the industry's standard dummy text ever since the 1500s,
                when an unknown took and pe specimen book.
            </p>
            <hr>
        </div>
    </div>
</div>
- CSS Snippet (home.css) 

.landing-header {
    background: #567890 url(../resource/bg_header.jpg) no-repeat;
    background-size: cover;
    min-height: 380px;
    text-align: center;
    padding-top: 200px;
    color: white;
    font-weight: 300;
}

.landing-header .landing-header-title {
    font-size: 40px;
}

.landing-info {
    text-align: center;
}

.landing-info .landing-info-title {
    font-size: 25px;
    font-weight: 300;
    text-align: center;
}

.landing-info hr {
    border-bottom: 1px solid #ddd;
    margin: 0;
}
  1. Interesting tip is that we can use these two classes: col-md-8 and col-md-offset-2 to make a centered <div> with in the current context.
  2. using background attribute to set the background color as well as the background image for the application home page, make sure the background image is wide enough to fill the whole window. Otherwise, the UI would be ugly.
