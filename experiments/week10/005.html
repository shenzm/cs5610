UI - Profile Page II

This experiment is based on the last experiments, it provide a right-col of the profile page, it is a tab navigation panel, which can be used to display more detailed user information.

Experiment Description:

  1. After logged in we can see the profile page of the user. The profile page consist of two part, we will focus on the right-col in this experiment.
  2. The right-col of the profile page is a tab panel which can be used to display multiple different content according to the active tab. I use AngularJS to implement this tab function.
  3. Username: aliceW (asd@asd.com)
  4. Password: asd

Experiment:

Click button on top-left corner to view in new page

Operation Analysis:

- HTML Snippet (profile.html) 

<div class="tab-bar">
    <ul class="nav my-nav-tabs" ng-init="activeTab = 'Profile'">
        <li role="presentation" ng-class="{active: activeTab == 'Profile'}">
            <a ng-click="activeTab = 'Profile'">Profile</a>
            <span class="menu-active">
                <span class="caret"></span>
            </span>
        </li>
    ...
    </ul>
</div>

<div class="tab-content">
    <div ng-switch="activeTab">
        <div role="tabpanel" ng-switch-when="Profile">
            <div class="panel panel-default">
                ...
            </div>
        </div>
        <div role="tabpanel" ng-switch-when="Favorites">Favorites</div>
        <div role="tabpanel" ng-switch-when="Friends">Friends</div>
    </div>
</div>
  1. Use ng-init to initialize a variable activeTab to "Profile", which will set the default tab to Profile tab.
  2. In the <ul> with class my-nav-tabs, pass an expression "{active: activeTab == 'Profile'}" to ng-class. which will set the class of the tag to active if the expression return true.
  3. And in the <div> with class tab-content, use ng-switch to switch between multiple sub <div> tags in which we set ng-switch-when variable accordingly.
