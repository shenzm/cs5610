Week7 - Experiment1

RockMongo - Web based MongoDB UI Manager

This experiment focus on manipulating/managing mongodb cartridge on openshift using another cartridge named RockMongo

Experiment Description:

  1. MongoDB does not include a GUI-style administrative interface. Instead most administration is done from command line tools such as the mongo shell. However some UI’s are available as separate community projects.
  2. RockMongo is one of these projects and it is supported on openshift. It's a web-based MongoDB management tool, written in PHP 5. We can use it to insert some sample data for our apps.
