Bootstrap - Modal SignIn Dialog

This experiment use Bootstrap's Modal utility to display a signin dialog that float above the home page when ever the sign link on the navigation bar is clicked.

Experiment Description:

  1. Ths experiment is on Home.html, which is shown in the following iframe.
  2. On default, the home page displays, and the signin link will be shown at the right of the navigation bar.
  3. If click on the Signin link, then a signin dialog would show in front of the home page, and use the home page as a background.
  4. User can then sign in using this dialog.

Experiment:

Click button on top-left corner to view in new page

Code Analysis:

- HTML Snippet
<ul class="nav navbar-nav navbar-right">
    <li>
        <a class="signin-a-tag" data-toggle="modal" data-target="#signinModal">Sign In</a>
    </li>
</ul>
...
<div id="signinModal" class="modal fade" tabindex="-1" role="dialog"
     aria-labelledby="myModalLabel" aria-hidden="true">
    <div class="modal-dialog">
        <div class="modal-content">
            <div class="modal-header">...</div>
            <div class="modal-body">...</div>
        </div>
    </div>
</div>
  1. Include the bootstrap.min.js, bootstrap.min.css and jquery.min.js
  2. Create a link with <a>, set its attribute data-toggle to model, and point the data-target to id #signinModal, so that the link would trigger the display of the <div> with id #signinModal
  3. Create a initially hidden div with class modal and fade, add div with class modal-dialog in it. And div with class modal-header and modal-body in it.
