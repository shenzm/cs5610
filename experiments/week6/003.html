Week6 - Experiment3

Openshift - AgularJS App

This experiment try to host a static Angular JS application on Openshift. This is based on WEEK5 Experiement3, which is single page business info favoriting app that support routing between different views.

Experiment Description:

  1. Ths app is hosted on Openshift and shown in the following iframe.
  2. The application is the same one as WEEK5-EXP3, which is a single page application. The first page lists all the restaurant, and user can choose some of them in the list to favorite them. And the favorited business would be added to the list, and be display in the other view, if user click the Favorite navigation link.

Experiment:

Click button on top-left corner to view in new page

Code Analysis:

- Javascript Snippet
app.use('/app', express.static(__dirname + '/app'));
  1. Using the express.static middleware in the app, to serve static content for the application from the “app” directory in the application directory. The "app" has an index.html, which will be defaultly rendered by the server if no other url postfix is provided.
  2. Other than that, we have to copy the whole directory from the WEEK5-EXP3, so as to support the Angular JS app running on the openshift.
