CSS - CodeBox Utility Class

Description:

This CodeBox class is defined in "styles.css" and is used to display code snippet inside a browser. It can work as an small utility for all other web pages.

Experiment Detail:

Following is an example of <div> that uses css class ".code-box". It has the following features:

(1) Maintains source code format

(2) Code box support bottom scroll-bar, if the window width is too small.

<p>Lorem Ipsum is simply dummy text ... PageMaker including versions of Lorem Ipsum.</p>

- Code Snippet:

<div class="code-box" >
    <pre>
        &lt;p&gt;
            Lorem Ipsum is simply dummy text ... PageMaker including versions of Lorem Ipsum.
        &lt;/p&gt;
    </pre> 
</div>

.code-box {
    padding: 10px;
    border: solid 1px #cccccc;
    background-color: #eeeeee;
    font-family: 'Courier New';
    overflow: auto;
}

.code-box pre {
    font-family: 'Courier New';
    margin: 0px;
    overflow: visible;
}
