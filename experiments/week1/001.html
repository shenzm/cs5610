HTML5 - New Semantic Elements

Description:

HTML5 offers new semantic elements to define different parts of a web page, These semantic elements clearly describes its meaning to both the browser and the developer. So this experiment go through some useful new semantic elements in HTML5, such as <nav> ,<details> and <figcaption>.

Sub-experiments:

1. This is a <nav>, which is used to define a navigation bar in HTML5

Google | Yahoo | Microsoft

- Code Snippet:

The code uses a <nav> to enclose a bunch of <a>, which are navigation links of the web page. 

<nav class="box">
    <a target="_blank" href="http://www.google.com">Google</a> |
    <a target="_blank" href="http://www.yahoo.com">Yahoo</a> |
    <a target="_blank" href="http://www.microsoft.com">Microsoft</a>
</nav>
