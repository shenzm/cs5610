CSS - Responsive Web Page

Description:

This web page "005.html" is the experiment itself. It applys a new CSS style file "responsive.css".

Experiment Detail:

Basic concept is the adjust the CSS rules according to the browser's window size. The following difference can be shown if adjust the width of the window:

(1) If window-width > 960: white backgroud color + largest font size + largest padding + align center

(2) If window-width in range 500~960: lightcyan backgroud color + smaller font size + smaller padding

(2) If window-width < 500: lightblue backgroud color + smallest font size + smallest padding

- Code Snippet:

Basic approach is to make use of "@media screen" property as well as the condition clause related to "min-width" and "max-width" property to define different css rule under different web browser width condition.

@media screen and (max-width: 400px) {
    .container {
        width: inherit;
    }
    .experiment-contents {
        background-color: lightblue;
    }
}

@media screen and (min-width: 400px) and (max-width: 960px) {
    .container {
        width: inherit;
    }
    .experiment-contents {
        background-color: lightcyan;
    }
}

@media screen and (min-width: 960px) {
    .container {
        width: 900px;
        margin: auto;
    }
}
