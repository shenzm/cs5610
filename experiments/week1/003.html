CSS - Unified Page Style

Description:

CSS experiment is applied to all experiment pages, which can provide an unified style across all experiements

Experiment Detail:

Following is the standard layout for all the experiement pages. CSS file "styles.css" are used across all experiment pages. Typically, those "experiments-" prefixes class are specified for all experiment pages.

- Code Snippet (.experiment-nav):

This is the CSS code for the navigation bar, it basically defines how the navigation link is going to change according to different action.

.experiment-nav {
    font-family: Georgia;
    font-size: 12px;
    color: #ffffff;
    background-color: #666666;
    padding: 10px;
}

.experiment-nav a {
    text-decoration: none;
    color: #ffffff;
    padding: 10px;
}

.experiment-nav a:hover {
    background-color: #444444;
}

.experiment-nav a:visited {
    color: #ffffff;
}
