Javascript - Simple Animation

Description:

This experiment uses the basic component in JS to adjust how the div with id "object" is rendered in the browser.

Experiment Detail:

Basically it make uses of event handler mechanism, such as "onclick" and "onchange" to run related Javascript functions, in which change the either background color or the size of the div with id "object". Usage:

1. Adjust the range bar, to see the change of text and object div.

2. Press "Start Animation" to see how the backgound-color changes with time.

Object
80 px

- HTML Code Snippet:

1. Add a <div> with id object, which is used for the animation.

2. Add a range type <div> to adjust the size of the object

3. Add a <button> to trigger the animation. 

<div id="object">Object</div>
<label for="size-range"> Adjust the size of the rectangle: </label>
<span id="size-display">80 px</span> 
<input id="size-range" type="range"
          min="80" max="200" value="80"
          onchange="changeSize()" />
<button onclick="runAnimation()">Start Animation</button>

- JS Code Snippet:

1. Function runAnimation() is called by clicking the <button>. It calls the other function changeColor() which sets the color of the object with the input color parameter, every 1000 miliseconds.

2. Function changeSize() is called by changing the range bar. It sets a new width and height for the object according to the input range value. 

function runAnimation() {
    startAnimation();
    setTimeout(changeColor, 0, "lightblue");
    setTimeout(changeColor, 1000, "lightseagreen");
    setTimeout(changeColor, 2000, "lightpink");
    setTimeout(changeColor, 3000, "lightsalmon");
    setTimeout(changeColor, 4000, "white");
    setTimeout(endAnimation, 4000);
}
function changeColor(newColor) {
    var obj = document.getElementById("object");
    obj.style.backgroundColor = newColor;
}
function startAnimation() {
    var obj = document.getElementById("object");
    obj.innerHTML = "Start";
}
function endAnimation() {
    var obj = document.getElementById("object");
    obj.innerHTML = "End";
}
function changeSize() {
    var sizeDisplay = document.getElementById("size-display");
    var obj = document.getElementById("object");
    var sizeRange = document.getElementById("size-range");
    var size = sizeRange.value;
    sizeDisplay.innerHTML = size + "px";
    obj.style.width = size + "px";
    obj.style.height = (size / 2) + "px";
}
