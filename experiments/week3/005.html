jQuery - foldable panel

Description:

This experiment uses jQuery to implement a simple foldable panel

Experiment Detail:

Basically it make uses of jQuery's slideToggle function to dynamically hide or show the panel by clicking the panel header.

Click to toggle panel
Hello world!

- HTML Code Snippet:

<div id="flip">Click to toggle panel</div>
<div id="panel">Hello world!</div>

- CSS Code Snippet: 

#panel, #flip {
    padding: 10px;
    font-size: 18px;
    text-align: center;
    color: #566675;
    border: solid 1px #cae0f4;
    border-radius: 3px;
}
#flip {
    background-color: #edf6fe;
}
#panel {
    background-color: white;
    height: 200px;
    display: none;
}

- JS Code Snippet: 

$(document).ready(function () {
    $("#flip").click(function () {
        $("#panel").slideToggle(500);
    });
    $("#stop").click(function () {
        $("#panel").stop();
    });
});
