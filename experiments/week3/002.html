jQuery - Draggable Wigget Tryout

Description:

This experiment uses jQuery utilities to adjust some of the html contents.

Experiment Detail:

Basically user can add public tags, add drag these tags from public tags box to my tag box

Public Tags:

My Tags:

  • Drag to Here

- HTML Code Snippet:

<input id="tag-input" />
<button id="tag-add">Add a tag</button>
<p>Public Tags:</p>
<div id="tag-box-1" class="tag-box"> 
    <ul> </ul> 
</div> 
<p>My Tags:</p>
<div id="tag-box-2" class="tag-box">
    <ul> 
        <li id="placeholder" class="tag">Drag to Here</li>
    </ul>
</div>

- JS Code Snippet: 

$(function () {
    $("#tag-box-2 > ul").sortable({
        placeholder: "sortable-placeholder"
    });
    $("#tag-box-2 > ul").droppable({
        drop: function () {
            $("#tag-box-2 > ul > #placeholder").remove()
        }
    });
    $("#tag-box-1 > ul > li").draggable({
        opacity: 0.35,
        connectToSortable: "#tag-box-2 > ul",
        addClasses: false
    });
    $("#tag-add").click(function () {
        var text = $("#tag-input").val();
        addTag(text);
    });
});
