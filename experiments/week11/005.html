Operation Analysis:

- HTML Snippet (profile.html) 

<div class="zhenmins-tab" ng-controller="ProfileController">
    <tabset>
        <tab ng-repeat="tab in tabs" heading="{{tab.title}}" active="tab.active">
            {{tab.content}}
        </tab>
    </tabset>
</div>
  1. Use customized tabset tag to enclose a tab item.
  2. Define a tab tag and set its ng-repeat attribute, so it can iterate through a data list.
- Javascript Snippet (ProfileController.js) 

var profileController = angular.module("profileController", []);

profileController.controller("ProfileController",
function ($scope) {
    $scope.tabs = [
            { title:'Dynamic Title 1', content:'Dynamic content 1' },
            { title:'Dynamic Title 2', content:'Dynamic content 2' }
        ];
    }
);
  1. In the profile controller simply set the value of $scope.tabs, so the html page can iterate through and render this tabs one-by-one.
