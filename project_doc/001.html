Project Overview

ReadiBook is a single-page web application for book lovers to share books with each others. It's developed with MEAN stack, a full-stack javascript Agile development web framework.

Following is a list of icon features supported by the application:

  • Books search: This function replies on google books API, which allows user to search for books by its title, author, publisher, subject and isbn, etc. User can either perform an approximate search by combined keywords or apply a specific search on certain book attribute. The results will be shown under the search section, within a page of maximum 10 books, each entry of which shows a sketch of the book. Moreover, user can navigate between pages, to find their target books.

  • Book Details: From the book search page, or other pages that contains a sketch of a book, user can always click the "new-window" button to open a new window with the detail information of the book. It shows fine-grained information of the selected book, along with a list of reviews related, as well as a list of users who has already favorite the book.

  • Review Books: Rather than just getting information about books, this application also allows users to provide their own piece of information to enrich the incoming source of information. User can express their opinion on certain book, by using the review submit form locates under the list of previous reviews to submit their own reviews.

  • Favorite Books: User can "Favorite" a book at various pages, such as book search page, book detail page and other users' profile page. After "Favorite" a book, this book will be shown along with personal information on the user's profile page. User can manage their favorite books on their profile page. And other users can also see what books you have favorite.

  • Follow Readers: By using the above features, users are likely to encounter with users who has similar reading taste with themselves or have interesting comments on books. User can follow these users because these "friends" ares a good source of reading. And its easy to monitor what they have favorite since user's friend will also be displayed in the user profile page.

Other than these representative features, the ReadiBook application also provides a basic set of general web-app features, such as user login/register, user password reset, profile management, friend management, etc.
