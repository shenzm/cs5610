User Register and User Login

"Sign Up" and "Sign In" links are presented on the navigation link. User can use these links to jump to login/register page. These page provides some useful functions such as reset password for the user, and validate user input.

  • Register: If click on "Sign Up" link, user will be redirect to a register page. Which require uses to provide their email, firstname, lastname, nickname and password. A password confirmation is provide to prevent user from making a typo in their first password input field. The email must not be pre-occupied by other account. Since email is used to uniquely identify a user in the application.

  • Login: If user already registered, they can user their email and password to log in. After successful logged in, In the login page, a link is provide for the user to reset their password if they have "forgot their password".
