User Profile Update

By clicking the pencil button under the thumbnail image on user profile can lead the user to the profile edit page. Where user can either upload a new thumbnail image, update their personal information of change their password.

  • Head Image Upload: User can choose a jpeg file, and upload it as a head image. The new image will be display after it successfully uploaded to AWS S3 bucket.
