Landing Page and Navigation Bar

Landing Page is the initial page for the application. It basically gives a brief introduction on the ReadiBook application and displays a book cover images carousel slider which indicate its function as well.

  • Carousel Slider: Since ReadiBook is a book sharing application for book lovers, the main frame of the landing page display a three image carousel slider that displays the cover page of popular books. These books are "DA Vinci Code", "Harry Potter" and "The lord of the rings"

  • Navigation Bar: Navigation bar is an utility shared across all views. It has a link for the book search page at the left corner, and signin/signup link at the right corner. The navigation is fixed at the top of the web page to make it easier to reach when user are browsing the web-app with a small screen device, such as their phone.

Notice that the styling of ReadiBook is based on bootstrap. And it uses the "flaty" theme of bookstrap.
