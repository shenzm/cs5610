User Profile

After logging into the application, user will be redirect to the profile page. In the profile, a small name card displays on the left column, and a tab consist of two sub-tab: favorite book tab and friends tab. These tabs shows either the sketch of books or users.

  • Name Card: The small name card displays on the left column, it shows the user's basic information such as nickname, firstname, lastname and email. And it can display a small picture of the user. This picture will initially set to a placeholder. But user can upload their picture in the profile edit page.

  • Favorite Tab: This tab is used to show what book the user have favorite, a sketch information will be displayed for a book, only show its basic information, such as title, author, category and rating.

  • Friend Tab: This tab shows what friends current user has already followed. A sketch information will be display for a book similar to the book sketch, only show its basic information, such as nickname and email.
