Technologies Used

As mention, this application is developed with MEAN stack framework. Here are the related technologies

  • MongoDB related: Mongoose, AWS S3

    Use MongoDB as our backend database, install and utilize mongoose module in nodejs to connect and communicate with database. And also use AWS S3 to store user uploaded images.

    AngularJS related: AngularJS, Bootstrap, Bootstrap UI, Flaty theme

    Use AngularJS to template, generate and control the front-end web page. Incorporate with bootstrap css/js. And add some third party AngularJS module such as "ng-file-upload". And adopt the open source "flaty" theme of bootstrap to make web app prettier.

    Express related:

    Use Express as the server-side application framework, which handles the incoming requests with user defined functions.

    NodeJS related: passportJS, bcrypt, nodemailer, etc.

    Use NodeJS as the backend server for the whole application. To support the applications authentication function and password reset function, add passportJS, bcrypt and nodemailer module.
