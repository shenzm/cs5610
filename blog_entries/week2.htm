Week2

On the second week, I focused on utilizing existing css frameworks on web page payout design. Following are the five experiments I have done:
1) Bootstrap - Fixed Nav Bar
2) Bootstrap - Carousel
3) CSS - Grid System
4) Bootstrap - Table
5) Bootstrap - Sign In Page

Bootstrap - Fixed Nav Bar

This experiment uses the Bootstrap Framework to implement an Responsive Navigation Bar which fixed at the top of the web page. This is really helpful since the user can make use of the fixed navigation bar all the time no matter what part of the web page they are reading, it save user a lot of time to going back to the top of the web page to look for the navigation links.

Bootstrap - Carousel

This experiment uses the Bootstrap Framework and some customized css to implement a carousel uilitliy than can display multiple images in turns. Although the images automatically changes over time, but user can still change the image by hand by clicking the left / right arrow.

CSS - Grid System

This experiment creates a grid system by imitating grid system of Bootstrap Framework. Basic idea is to define different versions of "col-1" to "col-12" for various screen size. These "col" class are prefixed with "xs"(extra small), "sm"(small), "md"(medium), "lg"(large), which means their rules are going to apply on certain screen size. we can use the css property "@media (min-width: )" to differentiate different size of the brwoser.

Bootstrap - Table

Since a table is a great way to display a series of data in the web page. I tried out how to use the Bootstrap Framework to create an well formed table.

Bootstrap - Sign In Page

A login page is also need for almost every website nowadays to gather users information, and display web page according to the login user. This experiment uses the Bootstrap Framework and some customized css to implement a login page. User can choose "Remember Me" options as well when they are logging in.

Summary :

It helperful to have a well defined CSS framework that we can make use of. It would be extremely wasteful to buld up the webpage's layout from scratch by hand.
